# Themes

A collection of WordPress themes.

## Sheen

**Sheen** is a bold, vibrant, multi-niche **Full Site Editing (FSE) block theme** for service businesses. It's the first theme in this repo.

- **Flagship niche:** cleaning services (SparkleWave demo)
- **Second niche:** car detailing (ApexShine demo) — dark / electric-blue style variation + demo template
- **Light + dark** style variations, switchable in the Site Editor
- **Motion:** Lenis smooth scroll, scroll-reveals, animated counters, image carousel, custom cursor, preloader, video hero
- **Enhanced form controls:** flatpickr datepicker + Select2 dropdown
- **Complete template set:** front-page, home, index, single, page, archive, search, 404 + designed About / Services / Contact / Car-detailing pages
- Fully client-editable — no code needed to change text, prices, or sections

### Install

Appearance → Themes → Add New → Upload → `sheen.zip` → Activate.
Then edit everything under **Appearance → Editor**.
