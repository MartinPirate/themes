Sheen is a bold, vibrant, multi-niche service block theme. Flagship niche: cleaning services, with a car-detailing style variation. Full Site Editing, light + dark styles, and fully client-editable.

== Description ==

Sheen is a modern Full Site Editing (FSE) block theme for service businesses — cleaning companies, auto detailing, and more. It ships an energetic design system (bold typography, vibrant palette, generous whitespace), light and dark style variations, ready-made section patterns (hero, services, pricing, testimonials, FAQ, booking CTA), and a full set of templates. Everything is editable in the Site Editor with no code.

== Installation ==

1. In your admin, go to Appearance > Themes > Add New > Upload Theme.
2. Upload the sheen.zip file and click Install Now.
3. Click Activate.
4. Edit content and styles under Appearance > Editor.

== Frequently Asked Questions ==

= Is Sheen a block theme? =
Yes. It uses Full Site Editing — templates are HTML block markup and styling lives in theme.json plus the Site Editor.

= How do I switch to dark mode or the car-detailing look? =
Appearance > Editor > Styles > Browse styles, then pick a variation.
